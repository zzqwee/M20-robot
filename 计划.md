# M20机器狗应用开发计划

## 一、项目概述

开发一款连接M20/M20 Pro机器狗的Web应用，获取设备信息并实现基本控制功能。

## 二、技术架构

### 2.1 通信协议
- **协议类型**: TCP/UDP
- **UDP端口**: 30000（用于状态上报接收）
- **TCP端口**: 30001（用于控制指令发送）
- **默认IP地址**: 10.21.31.103（可手动配置）

### 2.2 技术栈
- **前端框架**: React + TypeScript
- **UI设计**: Apple FrostyGlass风格（毛玻璃效果）
- **通信方式**: WebSocket与后端服务通信，后端处理TCP/UDP协议

## 三、功能模块

### 3.1 核心功能（优先级：高）
1. **设备连接与配置**
   - 手动输入/配置机器狗IP地址
   - 建立TCP连接
   - 连接状态显示

2. **设备信息获取**
   - 设备型号（Version）
   - 设备序列号
   - 设备状态（运行/休眠等）
   - IP地址、MAC地址
   - 网关、子网掩码、DNS
   - 设备时钟

3. **运动控制**
   - 前进/后退（X轴速度）
   - 左移/右移（Y轴速度）
   - 左转/右转（Yaw角速度）
   - 停止命令

### 3.2 扩展功能（优先级：中）
4. **LED控制**
   - 点亮/熄灭
   - 闪烁模式

5. **麦克风控制**
   - 开启/关闭
   - 音量调整

6. **摄像头控制**
   - 开启/关闭
   - 分辨率调整

7. **传感器数据读取**
   - 激光雷达距离
   - 姿态角度
   - 温度

8. **电池状态**
   - 电压
   - 电量百分比
   - 充放电状态
